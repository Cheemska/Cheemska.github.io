第三步

进入github 创建一个库

点击右上角的 + 按钮 选择 New repository 创建一个 <用户名>.github.io 的仓库
切记 仓库名格式必须为 <用户名>.github.io

选择Public 这样别人才可以看见 Description选填 只是介绍这个库而已
Add a README file可以不选
